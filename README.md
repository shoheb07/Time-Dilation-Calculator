# Time-Dilation-Calculator
Time Dilation Calculator is a physics application that calculates the relativistic time experienced by an observer moving at speeds close to the speed of light. The calculator uses Einstein’s Special Theory of Relativity to determine how time slows down for a moving object compared to a stationary observer.

# Time Dilation Calculator

## Overview

Time Dilation Calculator is a web application developed using HTML, CSS, and JavaScript that computes relativistic time dilation using Einstein's Special Theory of Relativity.

The application allows users to enter a proper time interval and a velocity expressed as a fraction of the speed of light, then calculates the corresponding dilated time and Lorentz factor.


## Features

- Time dilation calculation
- Lorentz factor calculation
- Responsive interface
- Instant results
- Physics-based computation
